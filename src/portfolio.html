WebApp per task Odoo

Progetto

Webapp per la gestione di task tramite API di Odoo.

Tecnologia

React

Contesto

Odoo consente una gestione completa dei progetti, inclusa la contabilità delle ore lavorate e la suddivisione in task. Tuttavia, ogni utente Odoo richiede un abbonamento mensile, con costi che aumentano significativamente anche con pochi utenti.

Obiettivo

Ridurre i costi derivanti da molteplici abbonamenti mensili, mantenendo la possibilità di tracciare le ore lavorate dai collaboratori.

Soluzione

Ho creato un modello custom "modello_collaboratore" all'interno di Odoo, includendo campi come Username, Password, Valore (€) e altri. La webapp sviluppata si interfaccia con questo modello tramite le API di Odoo, permettendo l'autenticazione tramite credenziali personali, il monitoraggio delle ore per ogni collaboratore e la reportistica necessaria. Pur con alcune limitazioni rispetto all'uso di utenti Odoo con abbonamento, questo approccio garantisce un significativo risparmio economico e soddisfa le funzionalità richieste.
